# StudyHub

StudyHub is a full-stack web application designed to help students manage their academic tasks, subjects, and study activities in one place.

---

## Features
- Task and assignment management  
- Subject organization  
- Simple and clean user interface  
- Database-driven content  
- Backend API for data handling  

---

## Technologies Used

### Frontend
- HTML  
- CSS  
- JavaScript  

### Backend
- Node.js  
- Express.js  

### Database
- MySQL  

---

## Project Structure

StudyHub/

- html         # Frontend pages
- css          # Stylesheets
- js           # Client-side scripts
- media        # Images and assets
- backend      # Node.js backend server
- database.sql # Database schema

---

## How to Run Project

1. Clone repository  
git clone https://github.com/Ruba146/StudyHub.git  

2. Run the backend  
cd backend  
npm install  
node server.js  

3. Setup the database  
Open `database.sql` in MySQL and run script to create database and tables  

4. Open the frontend  
Open `html/index.html` in your browser
